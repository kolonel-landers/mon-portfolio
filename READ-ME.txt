# Portfolio Personnel - Mr. LS

## Informations générales
- **Auteur** : Sandjon Landry
- **Date de création** : 28/10/2025
- **Dernière mise à jour** : 05/11/2025
- **Technologies utilisées** : HTML5, CSS3, Bootstrap 4, JavaScript, jQuery, Owl Carousel, Isotope, Lightbox, Formspree
- **Résumé / Objectif** : Portfolio personnel présentant mes projets, compétences et contact, conçu pour montrer mon travail en tant que développeur junior.

## Installation et configuration
### Prérequis
- Navigateur moderne (Chrome, Firefox, Edge)
- Connexion Internet (pour charger les librairies via Content Delivery Network (Réseau de Distribution de Contenu)).

### Étapes
1. Cloner le dépôt :  
   ```bash
   git clone https://github.com/kolonel-landers/mon-portfolio.git

2. Ouvrir le fichier index.html dans le navigateur

3. Aucun serveur ou configuration particulière nécessaire

4. Pour tester le formulaire de contact, vérifier que Formspree est bien configuré avec votre email


#### Structure du projet
/portfolio
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js

│   └── [Images du portfolio]
└── README.md
-> index.html : Page principale du portfolio
-> css/style.css : Styles personnalisés
-> js/main.js : Scripts JavaScript pour interactivité
-> img/ : Images utilisées dans le portfolio


##### Fonctionnalités
- Menu de navigation responsive
- Sections : À propos, Projets, Compétences, Contact
- Galerie d’images filtrable avec Isotope
- Carrousel d’images avec Owl Carousel
- Lightbox pour visualiser les images en grand
- Formulaire de contact fonctionnel via Formspree
- Design responsive (mobile, tablette, desktop)


###### Utilisation
- Naviguer avec le menu en haut de la page
- Cliquer sur les filtres dans la section Projets pour afficher certaines catégories
- Cliquer sur les images pour les voir en grand via Lightbox
- Remplir le formulaire de contact pour envoyer un message via Formspree


####### Problèmes connus et limitations
- Le formulaire de contact nécessite une configuration Formspree pour fonctionner
- Certains effets CSS peuvent légèrement varier selon les navigateurs
